Stress Detection Using Facial Cues

Considering the vast spectrum of negative effects induced by stress, as well as the need for a less intrusive means of assessing stress than traditional approaches. A deep learning model was constructed using publically accessible datasets to extract two prevalent facial features, namely emotions and facial action units, and evaluate their usefulness in detecting stress.

Meal Planner Assisted by Machine Learning

This project's goal was to develop a web application that gives users personalised meal suggestions based on their interests, nutritional goals, budget, and food allergies. An advanced reinforcement learning model was created to identify recipes that fit the user's needs in order to accomplish this. The necessary information was gathered from numerous recipe websites using a web crawler. Finally, a user-friendly interface was created to collect user input and successfully present the findings.
